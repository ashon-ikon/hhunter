# Usage Guide

This guide covers both the HCAD web app (primary tool) and the HAR snapshot CLI pipeline.

---

## Part 1 — HCAD Web App

Start the server:
```bash
source .venv/bin/activate
hcad-app
# → http://localhost:5000
```

---

### 1.1 Dashboard (`/`)

The landing page shows 6 heat map cards. Each card links to a full-page interactive choropleth map covering all Harris County ZIP codes.

| Map | Metric | What to look for |
|-----|--------|-----------------|
| Price / sqft | Median market value ÷ living area | Low-value ZIPs with rising adjacent ZIPs |
| YOY Change | % appreciation vs prior appraisal year | ZIPs appreciating faster than the median |
| Investor Activity | % of parcels owned by entities (LLC, LP, Trust, Corp, etc.) | High investor concentration = validated demand |
| Permit Surge | Recent permit rate (permits since 2023 ÷ total) | Renovation wave arriving = gentrification signal |
| Gentrification Score | 40% YOY + 30% permit rate + 30% investor % | Composite early-mover indicator |
| Flip Potential | Distressed stock score | Unrenovated + below-RCN parcels |

---

### 1.2 Heat Maps (`/map/<key>`)

Each map is a self-contained Leaflet.js choropleth generated by Folium.

**Keys**: `price`, `yoy`, `investor`, `permits`, `gentrification`, `flip`

#### Navigating the map
- Pan and zoom normally
- The two focus ZIPs (77021 and 77088) are highlighted with a blue dashed border
- Color legend is in the bottom-left corner
- The right-side insights panel shows:
  - Focus ZIP rankings and values
  - Top 5 ZIPs for this metric
  - Houston-wide median

#### ZIP insert card
Click any ZIP polygon to open a detail card on the left side showing **all 18 metrics** for that ZIP across every map — price, YOY, investor pct, permit rate, gentrification score, flip score, and the sub-components of each.

This card persists if you press `S` to snapshot.

#### Snapshot (press `S`)
Pressing `S` (or clicking the camera button) captures the current viewport — map tiles, insights panel, and any open ZIP insert card — and downloads it as a PNG file.

Press `Esc` to close the snapshot preview overlay without saving.

---

### 1.3 Regenerating Maps (`/regen`)

If you want to refresh the maps after re-running `hcad-ingest`:

```
http://localhost:5000/regen          ← regenerate all 6 maps
http://localhost:5000/regen/flip     ← regenerate one map
```

This re-runs all DuckDB scoring queries and rewrites the HTML files in `static/hcad_maps/`.

### 1.4 Parcel Deal Screener (`/screener`)

The screener lets you query all 1.6M HCAD parcels, filtered and scored for a specific deal type.

#### Deal types

| Type | Best for | Key scoring components |
|------|----------|----------------------|
| **Fix & Flip** | Distressed stock to renovate and resell | RCN gap (25%), unrenovated (20%), building age (20%), price discount (20%), ownership lag (15%) |
| **BRRR** | Buy-rehab-rent-refinance-repeat | Buy discount (30%), rehab upside (25%), building age (20%), ZIP momentum (15%), land optionality (10%) |
| **Buy & Hold** | Long-term appreciation plays | ZIP YOY (35%), entry discount (25%), value vintage (20%), ownership lag (20%) |
| **Land Play** | Teardown / redevelopment candidates | Land dominance (40%), structure age (30%), lot size (20%), area trajectory (10%) |
| **Wholesale** | Motivated-seller targeting | Price discount (30%), ownership lag (25%), neglect signal (25%), RCN gap (20%) |

Scores are 0–100. Click **"How is the score calculated?"** in the sidebar to see the component breakdown with proportional bars.

#### Filters

| Filter | Description |
|--------|-------------|
| ZIP Codes | Comma-separated or space-separated (e.g. `77021, 77088`) |
| Min Score | Filter out low-confidence results (default 40) |
| Min / Max Value | Market value range |
| Max Year Built | Focus on older stock (e.g. `1980`) |
| Min Years Held | Ownership lag filter (e.g. `10` for 10+ years same owner) |
| Limit | Max rows returned |

#### Results table

Each result row shows:
- **Score bar** in the deal type's color
- **Address** and HCAD account link (opens `search.hcad.org/GISMap` in new tab)
- **Living area** in sqft + **lot size** in both acres and sqft
- **Market value**, **$/sqft** vs ZIP median, year built
- **Owner name** and entity/individual badge
- **Signal bullets** — 2–4 human-readable reasons the parcel scored well

#### ZIP Market Pulse panel

When ZIPs are entered, a live panel appears at the top fetching current HAR.com data per ZIP:
- Active listing count and median list price
- Median $/sqft and median days on market
- Recent sold count and median sold price

This data is fetched from HAR.com's public API in real time and does not require authentication.

#### Export to CSV

Click **⬇ Export CSV** (or add `?fmt=csv` to the URL) to download the full result set as a CSV file including all scoring columns.

#### URL parameters

You can bookmark and share screener queries:

```
/screener?type=flip&zips=77021,77088&min_score=50&max_price=250000&max_year=1980&limit=25
/screener?type=land&zips=77021&min_score=60&fmt=csv
```

| Param | Values |
|-------|--------|
| `type` | `flip`, `brrr`, `buy_hold`, `land`, `wholesale` |
| `zips` | Comma-separated ZIP codes |
| `min_score` | 0–100 |
| `min_price` / `max_price` | Dollar amounts |
| `max_year` | Year built cutoff |
| `min_held` | Minimum years owned |
| `limit` | Max rows (default 50) |
| `fmt` | `html` (default) or `csv` |

### 1.5 CLI Screener

The screener also runs from the command line:

```bash
# Top flip deals in 77021 and 77088
hcad-screen --type flip --zip 77021 77088 --limit 25 --min-score 50

# Land plays with price cap, save to CSV
hcad-screen --type land --zip 77021 --max-price 200000 --csv land_deals.csv

# Wholesale targets: held 15+ years
hcad-screen --type wholesale --min-held 15 --min-score 55 --limit 50
```

```
Options:
  --type       flip | brrr | buy_hold | land | wholesale
  --zip        One or more ZIP codes (space-separated)
  --limit      Max results (default 25)
  --min-score  Min score 0-100 (default 40)
  --min-price  Min market value
  --max-price  Max market value
  --max-year   Max year built
  --min-held   Min years current owner has held
  --csv        Output path for CSV export
```

## Part 2 — HAR Snapshot Pipeline

The snapshot pipeline processes browser-captured MLS listing data from har.com into comparable analysis and deal rankings.

### Quick Reference

```bash
init-snapshot --label "Acres Homes 77091"
extract-har --snapshot snapshots/<id> snapshots/<id>/raw/har
normalize --snapshot snapshots/<id>
qa --snapshot snapshots/<id>
analyze --snapshot snapshots/<id>
visualize --snapshot snapshots/<id> --all --limit 20

# One-shot
pipeline --snapshot snapshots/<id>
```

---

### 2.1 Capturing Data (HAR Export)

1. Open har.com in Chrome/Firefox
2. Press `F12` → Network tab
3. Search for properties, apply your ZIP / type filters
4. Right-click the Network tab → **Save all as HAR**
5. Place the `.har` file in `snapshots/<id>/raw/har/`

Or use the direct API (no auth required):

```bash
python -m src.fetch_searchlistings \
  --snapshot snapshots/<id> \
  --zip 77088 \
  --for-sale 1        # 0 for rentals
```

---

### 2.2 Pipeline Steps

#### Extract
```bash
extract-har --snapshot snapshots/<id> snapshots/<id>/raw/har
```
Parses HAR files and writes `out/extracted/listings_raw.json`.

#### Normalize
```bash
normalize --snapshot snapshots/<id>
```
Coerces types, normalizes ZIPs, splits into `active.csv`, `sold.csv`, `rentals.csv`. Adds derived fields: `era_bucket`, `size_bucket`, `flip_box_flag`.

#### QA
```bash
qa --snapshot snapshots/<id>
```
Validates record counts, checks for missing fields, flags anomalies. Output: `out/qa/qa_report.json`.

#### Analyze
```bash
analyze --snapshot snapshots/<id>
```
Builds comparable cohorts and computes PPSF spreads (active ask vs sold median). Outputs:
- `out/analysis/ranked_candidates.csv` — all actives ranked by spread
- `out/analysis/scoreboard_segments.csv` — ZIP-level market metrics
- `out/analysis/streets_top.csv` — top streets by deal density

#### Grid Analysis
```bash
grid-analysis --snapshot snapshots/<id>
grid-analysis --snapshot snapshots/<id> --cell-size-m 400 --export-geojson
```
Bins parcels into ~400m grid cells and scores each cell by renovation spread, deal density, and market velocity.

### 2.3 Visualizing Results

```bash
# Available artifacts
visualize --snapshot snapshots/<id> --artifact ranked
visualize --snapshot snapshots/<id> --artifact scoreboard
visualize --snapshot snapshots/<id> --artifact grid_scoreboard
visualize --snapshot snapshots/<id> --artifact qa
visualize --snapshot snapshots/<id> --all --limit 20

# Custom column view
visualize \
  --snapshot snapshots/<id> \
  --artifact ranked \
  --columns address,zip,list_price,upside_to_p70,rank_score \
  --limit 15
```

**All artifact names:**
`ranked`, `scoreboard`, `streets`, `grid_scoreboard`, `grid_candidates`, `grid_streets`, `active`, `sold`, `rentals`, `requests`, `qa`, `normalize`

### 2.5 Rental Analysis

```bash
# Fetch both sales and rentals into same snapshot
python -m src.fetch_searchlistings --snapshot snapshots/<id> --zip 77088 --for-sale 1
python -m src.fetch_searchlistings --snapshot snapshots/<id> --zip 77088 --for-sale 0

normalize --snapshot snapshots/<id>

python << 'EOF'
import pandas as pd, statistics

sales   = pd.read_csv('snapshots/<id>/out/normalized/active.csv')
rentals = pd.read_csv('snapshots/<id>/out/normalized/rentals.csv')

med_price = sales['LISTPRICEORI'].median()
med_rent  = rentals['LISTPRICEORI'].median()

print(f"Median sale price:   ${med_price:,.0f}")
print(f"Median monthly rent: ${med_rent:,.0f}")
print(f"Gross cap rate:      {med_rent * 12 / med_price * 100:.2f}%")
print(f"Price-to-rent ratio: {med_price / med_rent:.1f}x")
EOF
```

## Tips

- **HCAD vs HAR**: Use the HCAD screener for deal sourcing (where to look, which accounts). Use HAR snapshots for comp validation and offer pricing (what things are selling for right now).
- **Focus ZIPs**: 77021 (South Main / Riverside Terrace) and 77088 (Acres Homes) are highlighted on every heat map. Run screener queries filtered to these ZIPs to find specific parcels matching your strategy.
- **Score legend**: The score breakdown in the screener sidebar tells you exactly what's driving a high score. A 75-point flip deal that scores high on "Ownership Lag" + "Unrenovated" is a very different conversation than one scoring on "RCN Gap" + "Price Discount".
- **ZIP pulse**: The HAR.com market data panel shows current active inventory and median sold prices. If HCAD says a parcel is at a 30% discount to RCN but HAR shows the ZIP median is already $180/sqft, the math works — but if HAR shows 90-day DOM, pace the exit accordingly.
